# QR Code Card Component

This project is a responsive QR Code Card Component created using HTML and CSS. The design features a clean card layout with a QR code, heading, and descriptive text, centered on the page using Flexbox.

## Features
- Responsive card design
- Clean and modern user interface
- Fully built with HTML and CSS
- Flexbox-based layout for easy centering
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was developed to practice front-end development skills, including layout design, responsive styling, and component-based UI creation.

## Learning Outcomes
- Structuring web pages with HTML
- Styling components with CSS
- Using Flexbox for alignment and positioning
- Creating responsive user interfaces
